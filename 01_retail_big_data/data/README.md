# Dataset

The original `Retailsales.csv` dataset is not included in this repository due to file size limitations.

The dataset is downloaded directly inside the notebook from a public cloud storage link.

Columns:
- product_id
- store_id
- date
- sales
- revenue
- stock
- price
- promo_type_1
- promo_bin_1
- promo_type_2
